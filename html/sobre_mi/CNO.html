Seguridad Informática CNO V

Duración: 1 semestre • Créditos: 6 • Modalidad: Presencial

Descripción

Esta asignatura introduce los principios, técnicas y buenas prácticas para proteger sistemas, redes y datos frente a amenazas y vulnerabilidades. Cubre conceptos teóricos y labores prácticas para la identificación, mitigación y respuesta ante incidentes de seguridad informática.

Objetivos

  • Comprender modelos y fundamentos de la seguridad de la información.
  • Identificar y evaluar vulnerabilidades en sistemas y redes.
  • Aplicar medidas de protección: criptografía, control de acceso y hardening.
  • Planificar y ejecutar respuestas a incidentes y recuperación ante desastres.

Portafolio Digital

El propósito principal del portafolio digital es la recopilación de evidencias a lo largo del curso, para de esa forma tener constancia de las competencias y habilidades adquiridas por el alumno. Además, el portafolio digital fungirá como un medio de pruebas para el alumno y una forma de demostrarse a sí mismo y al docente las capacidades adquiridas.

Información Técnica

Este portafolio digital está realizado en su mayoría con el lenguaje de hipertextos HTML y su complemento para estilos embebidos CSS. Mientras que posee algunos scripts realizados en JavaScript para las funcionalidades correspondientes a la interfaz de autorespuesta solicitada.

Adicionalmente, se debe mencionar que la plataforma de hosting utilizada es GitHub Pages, y este proyecto fue desarrollado con el uso en paralelo de las herramientas de GitHub Desktop y Visual Studio Code.
