Actividad 02

Casos de estudio sobre ciberataques

Lopez Lopez Josué Emmanuel

Introducción

El marco unificado sugerido para estructurar la seguridad dentro de las redes de comunicación en seguridad informatica es conocido como ITU-T X.800 que junto con el glosario oficial de la web RFC 4949 crean un marco de referencia para la gestión de seguridad informática., en esta actividad se analizaran varios casos de estudio bajo estas herramientas a medio de practica.

Desarrollo

Vector de Ataque

  • Vulnerabilidades conocidas dado el uso de software obsoleto (Windows Server 2003) y un sistema operativo sin soporte vigente (Windows 7)

  • Uso de un sistema legacy sin soporte y falto de actualizaciones de seguridad

  • CVE-2019-0604: Vulnerabilidad de Ejecución Remota de Código (RCE) en SharePoint. Falta de políticas de gestión de parches (Patch Management).

  • Defensa Perimetral Débil: Servidores expuestos a internet sin segmentación adecuada (DMZ) y posible falta de concientización de usuarios ante correos maliciosos.

Reflexiones del Alumno

En conclusión, los escenarios analizados muestran que los problemas de seguridad pueden tener distintos orígenes, desde errores humanos hasta ataques planeados, pero todos pueden afectar gravemente a las organizaciones. Más allá de la complejidad técnica, lo importante es reconocer que la prevención, la preparación y la respuesta rápida son claves para reducir el daño.
